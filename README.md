# Projeto TypeScript

Criando Projeto nodejs com typescript. configurando o projeto e criando template para iniciar novos projetos.

## Instalar Dependencias

```sh
  npm install
```

## Compilar Projeto

```sh
  npm run build
```

## Iniciar o Projeto

```sh
  docker run mongodb

  npm run start
```

## Iniciando com Docker

```sh
  docker-compose up -d
```
